## 動機

現今社會由於科技的進步和時代的發展，人們可以藉由各種3C產品透過連上網路來得到社會上的各種資訊。

然而隨著資訊便利的產生，所帶來的缺點就是讓人非常擔憂的資訊安全問題。例如電腦犯罪、電腦病毒等。因此，了解資訊安全重要性這件事在我們這個世代是一個重大課題。

因此我們想藉由CTF這種模式來了解資訊安全的知識和原理並從中學習相關的技術。
